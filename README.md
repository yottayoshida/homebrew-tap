# yottayoshida/homebrew-tap

Homebrew formulae for [yottayoshida](https://github.com/yottayoshida) projects.

## Usage

```bash
brew tap yottayoshida/tap
brew install <formula>
```

Or install directly:

```bash
brew install yottayoshida/tap/<formula>
```

## Available Formulae

| Formula | Description | Source |
|---------|-------------|--------|
| [lkr](Formula/lkr.rb) | LLM Key Ring — manage LLM API keys via macOS Keychain | [llm-key-ring](https://github.com/yottayoshida/llm-key-ring) |
